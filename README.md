# Duck Verse Arena

A playable browser fighting prototype by Rare Ink Studios.

## What is included

- Start screen
- How to play screen
- Super Duck vs Bat Duck
- Phantom Train Station arena
- Movement, jumping, punching, special attacks
- Health bars
- Win screen
- Desktop controls
- Mobile on-screen controls

## Run locally

Open `index.html` in a browser.

## Deploy on GitHub Pages

1. Upload these files to your Rare Ink Studios GitHub repo.
2. Go to Settings → Pages.
3. Choose Deploy from branch.
4. Select `main` and `/root`.
5. Save.

## Controls

Player 1:
- A/D move
- W jump
- F punch
- G special

Player 2:
- Arrow keys move/jump
- / punch
- . special

Mobile:
- Use on-screen controls.
