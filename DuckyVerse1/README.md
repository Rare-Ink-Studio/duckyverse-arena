# DUCKYVERSE

DUCKYVERSE is a playable browser fighting game prototype by Rare Ink Studios.

This version is a simple one-player game:

- You play as Hero Duck
- CPU plays as Rival Duck
- Wallet screen is included
- Guest mode is playable
- Desktop controls included
- Mobile touch controls included
- DUCKYVERSE poster image included
- Castle arena background included
